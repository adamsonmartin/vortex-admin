# VORTEX ADMIN – iGaming Operator Dashboard

Professional dark-themed admin interface built for online casino and betting operators.

### Features

- Real-time dashboard with key performance metrics (revenue, active players, RTP, total bets)
- Player management system with search and filtering
- Games overview and performance tracking
- Bonus and promotion management
- Responsible Gaming tools (deposit limits, reality check, self-exclusion)
- Clean, modern neon iGaming design

### Tech Stack

- HTML5
- Tailwind CSS
- Vanilla JavaScript
- Font Awesome

### Purpose

This project demonstrates my ability to design and develop complex, production-like admin panels for the iGaming industry. It showcases clean UI/UX, structured code, and understanding of real operator workflows.
